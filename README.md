## Prerequisites

- Node.js 18+
- pnpm

## Install

```bash
pnpm install
```

## Build

```bash
pnpm tsc
```

## Run

```bash
node dist/index.js
```

## Switch networks (optional)

Edit `src/index.ts` and change the HyperSync `url` to your target chain. See docs: `https://docs.envio.dev/docs/HyperSync-LLM/hypersync-complete`
